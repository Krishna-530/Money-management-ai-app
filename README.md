# 💰 SmartBudget AI

SmartBudget AI is a simple mobile app built to help students manage their daily expenses. The goal of this project is to make expense tracking easy, quick, and useful in everyday life.

---

## 🚀 Features

- Add and track daily expenses  
- Categorize spending (food, travel, etc.)  
- View total spending  
- Budget alerts and reminders  
- Basic insights on spending habits  
- Detect unusual expenses  
- Works offline  

---

## 🛠️ Tech Stack

- Flutter (Dart)  
- SQLite (local database)  
- Firebase Authentication  
- Notifications  

---

## ⚙️ How It Works

- User logs into the app  
- Adds daily expenses  
- Data is stored locally  
- App tracks and organizes spending  
- Alerts and insights are shown  

---

## 📌 Note

This project was developed as part of a college submission and focuses on solving a real-life problem in a simple and practical way.
